# Database Systems Projects

A collection of database design and SQL development projects completed as part of CSE 3153 - Database Systems at Kennesaw State University.

This repository demonstrates the design, implementation, and management of relational databases using SQL. The projects focus on database modeling, normalization, entity relationships, schema creation, and data management.

## Technologies

* SQL
* SQLite
* Relational Database Design
* Database Modeling
* Entity Relationship Diagrams (ERDs)
* Data Normalization

## Repository Structure

### Hospital Database Project

A relational database designed to model a hospital management system.

The database includes entities such as:

* Patients
* Employees
* Departments
* Treatments
* Beds
* Medical Tests

Key concepts demonstrated:

* Relational database design
* Primary and foreign keys
* Entity relationships
* Data integrity
* Schema creation
* Healthcare data modeling
